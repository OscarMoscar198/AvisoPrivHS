Aviso de Privacidad

Datos personales que recabamos y su finalidad

  • Datos de identificación: nombre completo.
  • Datos de contacto: números telefónicos, correo electrónico.
  • Datos sobre actividades físicas: registros de ejercicios, progreso en entrenamientos.
  • Datos sobre características físicas: edad, peso, estatura.

Finalidades del tratamiento

  • Monitorear el progreso en actividades físicas.
  • Personalizar la experiencia del usuario.
  • Proporcionar asistencia y recomendaciones personalizadas.

Transferencias de datos personales

Sus datos personales no serán transferidos a terceros sin su consentimiento, salvo aquellas que sean necesarias para atender requerimientos de información de una autoridad competente.

Medidas de Seguridad

En EntrenAT se han implementado diversas medidas de seguridad para proteger sus datos personales:

  • Encriptación de contraseñas mediante algoritmos de hashing robustos.
  • Validación de contraseñas durante el registro.
  • Uso de tokens de autenticación para sesiones de usuario seguras.
  • Validación de entradas en la API para prevenir inyecciones de SQL.

Identificación de Información Sensible

La información sensible manejada en EntrenAT ha sido clasificada como:

  • Alta prioridad: contraseñas.
  • Media prioridad: número telefónico, correo electrónico.
  • Baja prioridad: registros de ejercicios, progreso en entrenamientos, edad, peso, estatura.
